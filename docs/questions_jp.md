# StoryWorth 質問リスト（日本語訳）

出典: [Complete StoryWorth Questions List 2026 (+50 Bonus Prompts) — Memory Murals](https://www.memorymurals.com/journal/storyworth-questions-list)

各カテゴリには2種類の質問があります。
**StoryWorth風**は、StoryWorthの公開サンプル質問をもとに言い換えたもの。
**Memory Murals Vault**は、Memory Muralsが独自に作成したオリジナルの質問です。

-----

## 1. 幼少期（Early Life and Childhood）

### StoryWorth風

- いちばん古い記憶は何ですか？
- 育った家について教えてください。
- 子どもの頃の好きな食べ物は何でしたか？
- 好きだった遊びやおもちゃは何でしたか？
- あなたの寝室はどんな様子でしたか？
- 子どもの頃いちばん仲が良かった友達は誰でしたか？

### Memory Murals Vault

- あなたの一番最初の記憶は何ですか？
- 育った家について、どんなことを覚えていますか？
- 子どもの頃、家での typical な一日はどんな様子でしたか？
- 子どもの頃、何をするのが大好きでしたか？
- 初めての友達は誰で、一緒に何をしていましたか？
- 子どもの頃、何をすると叱られましたか？
- 幼い頃、何が一番怖かったですか？
- どんな匂い・音・味が、あなたを子ども時代に引き戻しますか？
- 子どもの頃に起きた、一番おかしかった出来事は何ですか？
- あなたの子ども時代について、人々に分かってほしいことは何ですか？

-----

## 2. 家族と育ててくれた人々（Family and the People Who Raised Them）

### StoryWorth風

- あなたのお母さんはどんな人でしたか？
- あなたのお父さんはどんな人でしたか？
- 祖父母から何を学びましたか？
- 一番よく覚えている家族の伝統は何ですか？
- あなたの家では、祝日はどんな様子でしたか？

### Memory Murals Vault

- あなたを育ててくれた人について、一番よく覚えていることは何ですか？
- 家族から受け継いで、今も持ち続けている習慣や言葉はありますか？
- あなたの家族が持っていて、大好きだった伝統は何ですか？
- 家族の誰かが、言葉にせずに教えてくれた教訓は何ですか？
- 家族の中で、あなたを最も形作ったのは誰だと思いますか？それはどのように？
- 若い頃には分からなかったけれど、今は家族に感謝していることは何ですか？
- まだ聞けるうちに、家族の誰かに聞いておけばよかったと思うことは何ですか？

## 3. 成長と自分探し（Growing Up and Finding Yourself）

### StoryWorth風

- 10代の頃、好きだった音楽は何でしたか？
- 大人になったら何になりたいと思っていましたか？
- 印象に残っている先生について教えてください。
- 高校を卒業した後、何をしましたか？
- 初めて持った車は何でしたか？

### Memory Murals Vault

- 若い頃に抱いていた、誰も知らなかった夢は何ですか？
- 「ようやく大人になった」と感じた瞬間はどんな時でしたか？
- 学生時代の一番強い記憶は何ですか？
- 10代の自分にすぐ引き戻してくれる曲や映画は何ですか？
- 若い頃に世界について信じていたことで、後に違っていたと分かったことは何ですか？
- 若い頃の経験で、人生の方向を変えたものは何ですか？

## 4. 愛と人間関係（Love and Relationships）

### StoryWorth風

- 配偶者とはどのように出会いましたか？
- 結婚式の日はどんな様子でしたか？
- 初めて二人で暮らした家はどんな所でしたか？
- 若いカップルにどんなアドバイスをしますか？

### Memory Murals Vault

- 初めて恋に落ちた時のことを、どんなふうに覚えていますか？
- 人間関係について受けた、一番のアドバイスは何ですか？
- 愛する人との小さな瞬間で、よく思い返すものは何ですか？
- あなたの人生で、最も予想外だった友情は誰とのものですか？

## 5. 仕事・目的・野心（Work, Purpose, and Ambition）

### StoryWorth風

- 初めての仕事は何でしたか？
- 何を生業にしていましたか？そして、なぜそれを選びましたか？
- これまでで一番大変だった仕事は何でしたか？
- 今でも誇りに思える仕事の思い出は何ですか？

### Memory Murals Vault

- 初めての仕事は何で、そこから何を学びましたか？
- これまでで最も誇りに感じた仕事は何ですか？
- 大人になったら何になりたかったですか？そして、それはどう変わりましたか？
- 仕事や人生の進路で、最も難しかった決断は何ですか？

## 6. 親であること・子育て（Parenthood and Raising a Family）

### StoryWorth風

- お子さんがそれぞれ生まれた日は、どんな様子でしたか？
- 親であることで、一番大変だと感じたことは何ですか？
- 子どもとの、決して忘れられない瞬間は何ですか？
- 子どもたちに、自分の子ども時代について何を覚えていてほしいですか？

### Memory Murals Vault

- 親になって、一番驚いたことは何ですか？
- 子どもとの瞬間で、あなたにとって最も大切だったものは何ですか？
- あなたの姿を見て、子どもたちが学んでくれたらと願うことは何ですか？
- 子どもがしたことで、涙が出るほど笑ったことは何ですか？

## 7. 困難な時期と回復力（Hard Seasons and Resilience）

### StoryWorth風

- 人生で一番つらかった時期はいつですか？
- どのようにして困難な時期を乗り越えましたか？
- 誰も知らなかったと思う、あなたが抱えていた葛藤は何ですか？

### Memory Murals Vault

- 人生で最もつらかった時期はいつで、どうやって乗り越えましたか？
- 今のあなたを形作った喪失や苦しみは何ですか？
- 誰も気づいていなくても、自分が最も強かったと感じたのはいつですか？
- 物事が不可能に思えた時、あなたを支えてくれたものは何ですか？

## 8. 喜び・楽しさ・明るい側面（Joy, Fun, and the Lighter Side）

### StoryWorth風

- 一番好きな旅行の思い出は何ですか？
- 決して忘れられない食事は何ですか？
- これまでで一番冒険的だったことは何ですか？
- 長年にわたって愛してきた趣味は何ですか？

### Memory Murals Vault

- これまでで一番おいしかった食事は何で、誰と一緒でしたか？
- これまでで最も冒険的だったことは何ですか？
- どんな時でも必ず笑ってしまうものは何ですか？
- 訪れた場所で、決して忘れられない所はどこですか？
- ほとんどの人が知らない、あなたの才能や趣味は何ですか？

## 9. 知恵・振り返り・遺産（Wisdom, Reflection, and Legacy）

### StoryWorth風

- 若い頃の自分にどんなアドバイスをしますか？
- 何に最も感謝していますか？
- 孫たちに、あなたについて何を覚えていてほしいですか？
- 人生が教えてくれた最も大切な教訓は何ですか？

### Memory Murals Vault

- 20歳の頃に知っておきたかったと思う、今知っていることは何ですか？
- 人生を振り返って、最も感謝していることは何ですか？
- 人々があなたについて誤解していると思うことは何ですか？
- あなたの人生のどの瞬間が、決して忘れられないでほしいと願いますか？
- 家族の未来の世代に、あなたについて知っておいてほしいことは何ですか？
- 愛する人たちに一つだけメッセージを残せるとしたら、何を伝えますか？
