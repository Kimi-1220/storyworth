# 製本（ハードカバー）調査メモ

最終的な成果物は「ハードカバーの伝記1冊」。小ロット（1冊から）で発注できる製本サービスの調査。

## 調査した選択肢

### 1. ガップリ！（gappri）

- https://www.gappri.jp/gallery/item/20000504.php
- 高品質な上製本（ハードカバー）。事例は立派だが、**1冊あたり5万円程度かかる見込み**で、
  ギフト商品の原価としては高すぎる。

### 2. 冊子印刷のオーダー（sasshi-insatsu）

- https://order.sasshi-insatsu.com/#6
- 価格シミュレーションあり。中間候補。

### 3. 小説製本（ネット印刷のN-pri） ← **採用候補（一択）**

- https://publish.n-pri.jp/spec/novel/
- 個人の小説同人誌向けの製本サービス。1冊からの小ロットで価格が現実的。
- 文庫・四六判など伝記に向いた判型がある。
- 参考記事（小説製本の体験記）: https://note.com/ryuon_novel/n/n0443a65abea0

## 結論（現時点）

**小説製本（N-pri等の同人誌・自費出版系印刷所）一択。**
ハードカバー専門の高級製本は1冊5万円クラスで原価が合わない。
小説製本系なら1冊数千円台に収まる想定で、年間プラン（製本込み）の価格設計が成立する。
