# 日本版 Storyworth 企画書

## サービス概要

親や祖父母の人生の物語を、毎週1つの質問に答えてもらうことで少しずつ集め、
最終的に1冊のハードカバー本（伝記）として製本して届けるサービス。

本家: [Storyworth](https://welcome.storyworth.com/)

## プロダクトの肝（最重要の2点）

このプロダクトの価値は、突き詰めると次の2点に集約される。

1. **いかに充実した回答をもらえるか**（LINEメインを想定）
   ただ質問を淡々と送るだけでは、語り手は回答する気にならない。
   回答にリアクションを返す、回答を深掘りする質問を重ねる、小説の進捗を共有して
   モチベーションを保つ、回答が止まったら贈り主（子・孫）にリマインドを依頼する——
   「聞き上手な相手と話している」体験をLINE上に作ることが本質。
   → 詳細: [line-experience.md](./line-experience.md)

2. **いかにクオリティの高い小説を書けるか**
   集めた回答（断片的なテキスト・音声・写真）を、読み物として面白い
   「一冊の小説（伝記）」に仕上げる編集・執筆パイプラインの品質が、
   最終的な顧客満足と口コミを決める。
   → 詳細: [novel.md](./novel.md)

この2点は連動している。深掘りの質が上がるほど小説の素材が豊かになり、
小説の進捗を見せるほど回答のモチベーションが上がる。

### 日本版ならではのポイント

- **LINEで質問が届き、LINEで答えられる**
  本家はメールベースだが、日本のシニア層のメール利用率は低く、LINEの普及率が圧倒的に高い。
  週1回、LINE公式アカウントから質問が届き、トークに返信するだけで回答が完了する。
- **回答形式は自由**: テキスト・写真・音声（録音）なんでもOK。
- **Webでも回答可能**: 長文を書きたい人、PCで編集したい人向けにWebの回答ページも用意。
- **1年後（または質問を回答し終えたら）、回答をまとめてハードカバー製本**して家族に届ける。

## ターゲット

- **購入者（ギフター）**: 30〜50代。親・祖父母へのプレゼント（還暦・古希・母の日・敬老の日など）。
- **語り手（ストーリーテラー）**: 60〜90代の親・祖父母。LINEは日常的に使っているが、PC操作は苦手な層。

## ユーザーフロー

1. 子・孫がギフトとして購入し、語り手（親・祖母）を招待する
2. 語り手がLINE公式アカウントを友だち追加し、連携コードでアカウントを紐付け
3. 毎週決まった曜日・時間に、LINEで質問が1つ届く
   - 質問は家族がリストから選ぶことも、おまかせ（自動選択）にすることもできる
4. 語り手はトークに返信して回答（テキスト / 写真 / 音声）
   - Webの回答ページから書いてもよい
5. 家族はWebダッシュボードで回答を閲覧でき、追加の質問や感想を送れる
6. 規定の質問数（例: 52問 = 1年分）に達したら、本の編集・プレビュー → 製本注文
7. ハードカバーの伝記が自宅に届く

## 価格イメージ（仮・要検証）

| 項目 | 参考 |
| --- | --- |
| 本家 Storyworth | $99/年（1冊込み、追加 $79/冊） |
| 日本版想定 | 年間プラン 15,000〜25,000円（製本1冊込み）あたりを検討 |

製本原価の調査は [printing.md](./printing.md) を参照。
小ロット（1冊〜）のハードカバー製本は1冊あたり数千円〜5万円と幅が大きく、
**小説製本系のサービス（1冊数千円台）を使うのが現実的**という調査結果。

## なぜ今か

- 高齢の親世代のLINE利用が当たり前になった（メール不要でリーチできるのは日本特有の好機）
- 音声認識・LLMによる文字起こし・文章整形で「話すだけで本になる」体験が低コストで実現可能
- 終活・エンディングノート市場の拡大
